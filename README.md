# 🍃 Team 7 Naruto - Landing Page

[![Framework: Next.js](https://img.shields.io/badge/Framework-Next.js-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Styling: Tailwind CSS](https://img.shields.io/badge/Styling-Tailwind%20CSS-06B6D4?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

Sebuah aplikasi web *landing page* interaktif dan responsif yang didedikasikan untuk **Team 7** dari serial anime Naruto (Naruto Uzumaki, Sasuke Uchiha, Sakura Haruno, dan Kakashi Hatake). Proyek ini dibangun sepenuhnya menggunakan **Next.js** untuk performa kilat layaknya *Flying Raijin Jutsu*!

---

## 🚀 Fitur Utama

* **🥷 Profil & Statistik Karakter:** Menampilkan informasi detail, keahlian khusus (*jutsu* utama), dan statistik kemampuan masing-masing anggota Team 7.
* **📱 Desain Responsif & Modern:** Tampilan web yang adaptif dari layar *smartphone* hingga desktop.
* **⚡ Performa Optimal:** Memanfaatkan fitur *Image Optimization* bawaan Next.js untuk memuat aset gambar karakter tanpa lemot.
* **🎨 Estetika Anime:** Skema warna dan elemen visual yang dikustomisasi khusus agar terasa kental dengan atmosfer Konohagakure.

---

## 🛠️ Teknologi & Tools

* **Core Framework:** Next.js
* **Styling:** Tailwind CSS / CSS Modules *(Sesuaikan)*
* **Icons:** Lucide React / React Icons *(Sesuaikan)*
* **Deployment:** Vercel
